# Projeto AWS CDK 

## Objetivo do projeto 
Criar 2 aplicações, uma para CRUD de produtos, outra para monitorar os eventos da primeira API. 

## Tecnologias e conceitos usados 
- Spring Boot 
- Docker 
- AWS 
- Maven 

## Serviços utilizados da AWS 
- CDK (Para a criação de recursos ) e Cloud Formation
- ECS 
- RDS 
- AWS
- SNS 
- SQS 
- DynamoDB 
- S3
